    My Time - page description

    This page will contain the time sheets of the logged in user/tech

    The time sheets can be used as reference for the "real" time sheets, can be submited online to the designated person to be approved or can be saved/printed as .PDF files

    The data format of the time sheet will resamble with the current time sheet used by the company

    The time sheets data will be kept in a specialy created database so it can be queried at any time for any job, day, week, month or year since the date when the user/tech signed-up with the iChubb project
